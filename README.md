# Data Structure Project

## QuickStart

1. Build the c++ program. 

``` shell
mkdir build && cd ./build
cmake .. && make

```

2. Start c++ process server and web server.

```shell
./bin/process
node app.js
```


3. Open `http://localhost:3000/map` at browser.
